# Blinkit Sales Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes Blinkit's sales performance, customer ratings, outlet performance, and product categories.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Key KPIs
- Total Sales
- Average Sales
- Number of Items Sold
- Average Rating

## Insights
- Sales by Outlet Type
- Sales by Outlet Size
- Sales by Location
- Item Category Performance
- Customer Rating Analysis

- Power BI Dashboard (.pbit)
